📂 Estructura del proyecto

  • index.html — sitio público principal
  • admin.html — panel de administración (cambiar precios, renombrar productos, ver pedidos)
  • privacidad.html · terminos.html — páginas legales
  • manifest.webmanifest + service-worker.js — PWA
  • icons/ — íconos del logo (PNG + SVG)
  • img/ — fotos reales de productos (subir aquí)
  • tools/generar-iconos.html — regenerador de íconos PWA
  • css/styles.css — diseño completo
  • js/products.js — 2797 productos reales (de Valery)
  • js/main.js, cart.js, checkout.js, customers.js, admin.js, config.js — lógica

🔑 Credenciales por defecto

  • Administrador: admin / aluvima.admin.2026 — acceso total al panel
  • Propietario: propietario / aluvima.2026 — sin secciones admin

⚠ Cambia las contraseñas antes de publicar la app en Play Store.

💡 Para el día a día

  • Cambiar la tasa BCV: Admin → Configuración → Tasa de cambio
  • Renombrar productos: Admin → Renombrar productos (asigna nombres comerciales)
  • Ver pedidos: Admin → Pedidos
  • Cambiar WhatsApp: Admin → Configuración → Contacto
  • Activar/desactivar métodos de pago: Admin → Métodos de Pago
  • Backup completo: Admin → botón Backup arriba a la derecha
